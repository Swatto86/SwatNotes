Getting Started

QuickNotes is a production-grade desktop notes application built with Rust and Tauri. It features rich text editing, image support, automatic backups, and more.

Features

  • Rich text editing with images and attachments
  • Automatic backups and restore
  • Reminders with notifications
  • System tray integration
  • Global hotkeys
  • Beautiful themes powered by DaisyUI
